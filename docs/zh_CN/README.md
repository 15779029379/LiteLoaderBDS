## 🎨 项目简介
`LiteLoaderBDS`是一个基岩版官方服务端`Bedrock Dedicated Server`（以下简称 **BDS**）插件框架，提供强大的跨语言脚本插件支持能力和稳定的开发API支持。

👉[点击此处](https://lxl.litebds.com/#/?id=main)👈 查看对LL项目更详细的解释说明
